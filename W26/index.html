PICT B.E. Course Structure

Pune Institute of Computer Technology

Bachelor of Engineering (B.E.) in Computer Engineering curriculum overview. Explore the four-year course structure below.

First Year Engineering (FE)

  • Semester 1

  • Engineering Mathematics I

    Calculus, differential equations, and matrix algebra with engineering applications.

  • Engineering Physics

    Fundamental physics concepts with focus on electromagnetism and optics.

  • Engineering Chemistry

    Basic chemistry concepts relevant to engineering applications.

  • Basic Electrical Engineering

    Introduction to electrical circuits, devices, and basic electrical engineering principles.

  • Engineering Mechanics

    Statics and dynamics principles in engineering context.

  • Semester 2

  • Engineering Mathematics II

    Advanced calculus, numerical methods, and probability concepts.

  • Basic Electronics Engineering

    Fundamentals of electronic components, circuits, and devices.

  • Engineering Graphics

    Technical drawing and design visualization techniques.

  • Programming and Problem Solving

    Introduction to computer programming using C language.

  • Communication Skills

    Technical writing and presentation skills for engineers.

Second Year Engineering (SE)

  • Semester 3

  • Discrete Mathematics

    Mathematical structures for computer science including sets, logic, and graph theory.

  • Data Structures and Algorithms

    Implementation and analysis of fundamental data structures and algorithms.

  • Digital Electronics and Logic Design

    Boolean algebra, combinational and sequential circuits design.

  • Computer Organization

    Computer architecture, processor design, and memory systems.

  • Object-Oriented Programming

    OOP concepts with implementation in Java/C++.

  • Semester 4

  • Engineering Mathematics III

    Transform techniques, numerical methods for engineering problems.

  • Computer Graphics

    Principles and algorithms for 2D and 3D graphics rendering.

  • Microprocessor Architecture

    Architecture and programming of microprocessors.

  • Database Management Systems

    Relational database design, SQL, and transaction management.

  • Software Engineering

    Software development life cycle, requirements engineering, and design patterns.

Third Year Engineering (TE)

  • Semester 5

  • Theory of Computation

    Automata theory, formal languages, and computational complexity.

  • Systems Programming and Operating System

    Process management, memory management, file systems, and system programming.

  • Design and Analysis of Algorithms

    Algorithm design techniques and complexity analysis.

  • Computer Networks

    Network architecture, protocols, and security fundamentals.

  • Artificial Intelligence

    Search algorithms, knowledge representation, and intelligent agents.

  • Semester 6

  • Web Technology

    Client-side and server-side web application development.

  • Data Science and Big Data Analytics

    Data mining, statistical analysis, and big data processing frameworks.

  • Information Systems and Engineering Economics

    Systems design and economic aspects of engineering projects.

  • Software Modeling and Design

    UML, design patterns, and software architecture.

  • Mobile Application Development

    Android/iOS app development and mobile computing principles.

Final Year Engineering (BE)

  • Semester 7

  • High Performance Computing

    Parallel computing architectures, algorithms, and optimization techniques.

  • Machine Learning and Applications

    Supervised and unsupervised learning algorithms and practical applications.

  • Information and Cyber Security

    Cryptography, network security, and cybersecurity management.

  • Elective I

    Specialized area of computing (Cloud Computing, IoT, etc.)

  • Capstone Project Phase I

    First phase of major project involving research and preliminary design.

  • Semester 8

  • Distributed Systems

    Principles of distributed computing, algorithms, and systems design.

  • Software Testing and Quality Assurance

    Testing methodologies, tools, and quality management processes.

  • Elective II

    Advanced specialized area (Natural Language Processing, Computer Vision, etc.)

  • Professional Practices

    Ethics, intellectual property rights, and professional responsibility.

  • Capstone Project Phase II

    Implementation and evaluation of the final year project.
